### Map

| Syntax | Description |
| ----------- | ----------- |
| Leader | `<space>` |
| Exit | `' Leader q ' or ' Leader x '` |
| Comment | `' Leader c'` |
| NerdTree Open | `' Leader nt '` |
| Markdown Preview | `' Leader md '` |
| Markdown Preview Stop | `' Leader mds '` |
